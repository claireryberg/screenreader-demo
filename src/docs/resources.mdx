| Description                | Command                   |
| -------------------------- | ------------------------- |
| Start Voiceover            | Comman-fn-F5              |
| Start reading              | VO-A                      |
| Increase/decrease speed    | VO-Command-shift-arrow    |
| Open web rotor             | VO-U                      |
| Open the Commands menu     | VO-H-H                    |
| Skip all your browser tabs | VO-Command-fn-Right Arrow |
| Show/Hide Braille window   | VO-Command-fn-F9          |
| Move to Dock               | VO-D                      |
| Open Voiceover utility     | VO-F8                     |
| Lock/unlock Voiceover keys | VO-;                      |
| Jump to top of page        | VO-fn-Left Arrow          |
| Jump to end of page        | VO-fn-Right Arrow         |
